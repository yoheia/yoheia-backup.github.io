# lenovo x250 の設定メモ

### Windows 7 の設定

- タッチパッド
	- 起動時に F1 を押下し、BIOS 設定画面に移動する
	- [Config] - [Trackpad] で Disabled を選択
	- F10 を押下して保存する

- ページングファイルを使用しない
	- [コントロールパネル] - [システム] - [詳細設定] - [パフォーマンス] - [詳細設定] - [仮想メモリ]
	- [全てのドライブのページングファイルのサイズを自動的に管理する] のチェックを外す
	- [ページングファイルなし] を選択
	- 再起動する

- スクリーンセーバー
	- [コントロールパネル] - [個人設定] - [スクリーンセーバー]
		- ブランク
		- 待ち時間: 5分

- タスクバーのアイコンのサイズ
	- タスクバーと[スタート]メニューのプロパティ - [タスクバーのデザイン]
	- [小さいアイコンを使う] にチェック
